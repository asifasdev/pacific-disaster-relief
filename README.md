# Pacific Disaster Relief Coordination App

A lightweight app to help coordinate **requests**, **supplies**, and **volunteers** during disaster events (cyclones/floods), designed for simple workflows and low-bandwidth contexts.

## MVP Features
- Create disaster events
- Track requests (category, urgency, location, status)
- Track volunteers and assign to requests
- Track supplies and allocate to requests
- Simple dashboard: high urgency + low stock

## Repo Structure
- `backend/` FastAPI API
- `frontend/` React + TypeScript UI
- `docs/` planning docs (user stories, risks, data model)
